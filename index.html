Test Analysis


Accuracy Trends

Accuracy shows how many of the attempted questions in the test are correct.


  • Increasing accuracy shows less incorrect attempts are made.

  • Decresing accuracy indicates higher number of incorrect attemptsneed more practice to improve accuracy.

  • Single test can't be used for identifying future accuracy trends (please attempt more test).

Weak Areas

Score is the best parameter to define the weak areas of the student, it also shows on what topic student need to focus on.


  • Continuous increase in the score indicates strong areas.

  • Decrese in score indicates weak areas and need more practice to improve.

  • Single test can't be used for identifying weak and strong areas (take more test).

Improvement Trend

Mistakes corrected during test shows the improvement in knowledge of the student in the particular topic.


  • Continuous increase in mistake correction indicates strong command on the topic.

  • Decrese in mistake correction indicates lack of knowledge (Lack of Improvement).

  • Single test can't be used for identifying improvement trends (take more test)..

Performance Trends

Ranking in multiple test shows the performance of the student.


  • Continuous increase in rank shows nice performance.

  • Decrese in rank indicates poor performance (Need to Improve).

  • Single test can't be used for predicting precise rank in future (take more test).
